# my [adventofcode2025](https://adventofcode.com/) answers

## Dependencies
- Python >= 3.13  but should work on any version

## Installation
### Manual Installation

```bash
git clone https://github.com/Napstablook0/advent_of_code_answers.git
cd advent_of_code_answers
```

## Usage
```bash
py fileName.py
```
The answers will be printed in terminal
